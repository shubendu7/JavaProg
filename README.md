## About the Project
**JavaProg** is a collection of **practical Java programs** designed to help learners strengthen their understanding of key concepts in Java programming. 
This repository contains multiple sample programs demonstrating essentials like input/output operations, control structures, loops, functions, and basic logic building.

Each program is organized in a clear and beginner-friendly style, making this repository ideal for students and developers who are learning Java or preparing for academics and interviews.

---

## Concepts Covered
The programs in this repository showcase:
- Basic input/output (`Scanner`, `System.out`)  
- Conditional logic (`if`, `switch`)  
- Looping constructs (`for`, `while`, `do-while`)  
- Functions and method calls  
- Array usage and manipulation  
- Simple problem solving
